# Synthesized homepage section sequence

Blends Loyverse (feature tabs + industries + testimonials carousel) and SalesPlay (alternating two-column feature blocks dominant) into an order suited to astanabiz POS:

1. **Hero** — Two-column. Left: headline + sub + dual CTAs. Right: BrowserChrome with dashboard mock. Large.
2. **Trust stats** — Centered text counter band. 3–4 metrics. Small.
3. **Approach grid** — 6-card icon grid (Cloud, Offline, Inventory, Employee, Financial, Marketing). Medium.
4. **Feature: Sales Analytics** — Two-column, text left + BrowserChrome (chart) right. Medium.
5. **Feature: Inventory Management** — Two-column reversed, BrowserChrome (low-stock list) left + text right. Medium.
6. **Feature: Multi-store** — Two-column, text left + BrowserChrome (store comparison) right. Medium.
7. **Brand pillars** — 3-column cards (Astana POS / Astana Biz / MCBIZ). Medium.
8. **Industries** — 6-card grid with feature blurbs. Medium.
9. **Testimonials** — 3-column quote cards. Medium.
10. **Final CTA** — Centered partnership CTA on dark night gradient. Medium.

BrowserChrome is reused in steps 1, 4, 5, 6 with different placeholder content (dashboard / chart / inventory list / store grid).
